# WODX 화면 흐름도

## 전체 흐름

```
[스플래시 /]
    │
    └── START 버튼 클릭
            │
            ▼
    [로그인 /login]
            │
            ├── 신규 유저 or Firestore 프로필 없음
            │           │
            │           ▼
            │   [온보딩 /onboarding/profile]
            │           │ 이름/성별/몸무게/키/단위 입력
            │           │ 저장
            │           ▼
            │         [홈 /home]
            │
            └── 기존 유저
                        │
                        ▼
                      [홈 /home]
```

---

## 홈 (`/home`)

```
[홈 /home]
    │
    ├── 상단 헤더
    │       ├── 알림 벨 클릭 → [알림 패널] (슬라이드)
    │       │           ├── 알림 목록 (실시간)
    │       │           └── 읽은 알림 삭제
    │       │
    │       └── 햄버거 메뉴 클릭 → [사이드 메뉴] (슬라이드)
    │                   ├── 페이지 이동 링크
    │                   ├── 관리자 전용: WOD 관리 링크
    │                   └── 로그아웃
    │
    ├── 오늘 WOD 카드
    │       ├── 관리자
    │       │       └── ··· 드롭다운 → 수정 /admin/wod/edit/[id]
    │       │                        → 삭제
    │       │
    │       └── 일반 유저
    │               └── Done 버튼 → [기록 등록 /record/add?wodId=xxx]
    │
    └── 출석 달력 (이번 달)
```

---

## WOD (`/wod`)

```
[WOD /wod]
    │
    ├── 날짜 네비게이션
    │       ├── ← Prev 버튼 → 전날 WOD
    │       ├── 날짜 클릭 → DatePicker → 날짜 선택
    │       └── Next → 버튼 (다음날 WOD 있을 때만 활성화)
    │
    ├── WOD 카드
    │       ├── 파트별 타입/무게/팀여부/메모 표시
    │       ├── 관리자: 수정/삭제 드롭다운
    │       └── 일반 유저: Done 버튼 → [기록 등록]
    │
    └── 댓글 섹션
            ├── 댓글 목록 (좋아요/삭제)
            │       └── 좋아요 클릭 → 댓글 작성자에게 알림
            └── 댓글 입력 → 등록
```

## 기록 등록 (`/record/add`)

```
[기록 등록 /record/add?wodId=xxx]
    │
    ├── WOD 타이틀 표시
    │
    ├── 파트 선택 (A / B / C)
    │       └── 각 파트 독립적으로 입력 가능
    │
    ├── 레벨 선택 (Beginner / Scale / R'xd / Athlete)
    │
    ├── WOD 타입별 입력 폼
    │       ├── For Time
    │       │       ├── 완주: 완료 시간 (분/초)
    │       │       └── DNF: 라운드 + 렙수 (또는 렙수만)
    │       │
    │       ├── AMRAP
    │       │       ├── 라운드 + 렙수
    │       │       └── 총 렙수만 옵션
    │       │
    │       ├── EMOM
    │       │       ├── Only Reps 체크 → 총 렙수 입력
    │       │       └── 미체크 → 실패 횟수 입력
    │       │
    │       └── Every / Strength
    │               └── 메모 입력
    │
    ├── 무게 입력 (공통)
    │       ├── 권장 무게 참고 표시
    │       └── 실제 사용 무게 추가/삭제
    │
    ├── 팀 WOD (파트에 팀 설정된 경우)
    │       ├── 파트너 이름
    │       └── 파트너 다른 무게 사용 여부
    │
    └── Submit 버튼
            └── 모든 파트 저장 → [기록 메인 /record]
```

---

## 기록 수정 (`/record/edit/[id]`)

```
[기록 수정 /record/edit/[id]]
    │
    ├── 진입한 파트만 활성화 (다른 파트 비활성화)
    │
    ├── 기존 기록 불러와서 폼 초기화
    │
    ├── 기록 등록과 동일한 입력 폼
    │
    └── Update 버튼
            └── 해당 파트 기록 업데이트 → [기록 메인 /record]
```

---

## 기록 메인 (`/record`)

```
[기록 메인 /record]
    │
    ├── 출석 달력
    │       └── 날짜 클릭 → 해당 날짜 기록 표시
    │
    ├── 레이더 차트 (Athlete Profile)
    │       └── 5가지 능력치 점수 표시
    │
    └── 탭
            ├── [내 기록 탭]
            │       ├── 선택한 날짜의 기록 카드
            │       │       ├── 수정 버튼 → [기록 수정]
            │       │       └── 삭제 버튼
            │       │
            │       └── 기록 없을 때
            │               └── + 기록 추가 버튼
            │                       ├── WOD 있으면 → [기록 등록]
            │                       └── WOD 없으면 비활성화
            │
            ├── [리더보드(남) 탭]
            │       ├── 파트 선택 (A / B / C)
            │       └── 레벨별 섹션 (Athlete / R'xd / Scale / Beginner)
            │               └── 순위 / 이름 / 레벨 / 기록
            │
            └── [리더보드(여) 탭]
                    └── 리더보드(남)과 동일 구조
```

## PR Board (`/prdata`)

```
[PR Board /prdata]
    │
    ├── 카테고리별 아코디언 (접기/펼치기)
    │       │
    │       └── 운동 카드
    │               ├── 최고 기록 표시
    │               ├── 카드 클릭 → 히스토리 + 라인 차트 표시
    │               │       └── 히스토리 기록 삭제
    │               │
    │               └── + PR 추가 버튼
    │                       ├── 오늘 기록 없음 → [PR 입력 모달]
    │                       └── 오늘 기록 있음 → [교체 확인 모달]
    │                                               ├── 취소
    │                                               └── 교체하기 → [PR 입력 모달]
    │
    └── [PR 입력 모달]
            ├── 날짜 선택 (미래 날짜 불가)
            ├── 카테고리별 입력
            │       ├── Power / Strength → 무게 (kg / lb)
            │       ├── Skill → Max Reps (Hand Stand Walk 는 거리)
            │       └── Endurance / Conditioning → 시간 (시/분/초)
            │
            ├── 취소 or 바깥 클릭 → 닫기
            └── 저장 (값 입력 시 활성화)
                    └── 같은 날짜 기록 있으면 업데이트
                        없으면 새로 추가
```

## 프로필 (`/profile`)

```
[프로필 /profile]
    │
    ├── 보기 모드
    │       ├── 프로필 이미지 + 이름 + 이메일
    │       ├── 성별 / 몸무게 / 키
    │       └── 수정 버튼 → 수정 모드
    │
    ├── 수정 모드
    │       ├── 이름 / 성별 / 단위 / 몸무게 / 키 수정
    │       ├── 취소 → 보기 모드
    │       └── 저장 → 보기 모드
    │
    └── 로그아웃 → [스플래시 /]
```

---

## 관리자 WOD 등록 (`/admin/wod`)

```
[WOD 등록 /admin/wod]
    │
    ├── WOD 제목 / 날짜 입력
    │
    ├── 파트 설정 (최대 3개)
    │       ├── 타입 선택
    │       ├── 무게 추가/삭제 (도구/남성무게/여성무게/단위)
    │       ├── 팀 WOD 설정 (인원수)
    │       └── 파트 메모
    │
    ├── 전체 공지/메모
    │
    └── WOD 등록 버튼
            └── 등록 완료 → 모든 유저에게 알림 전송 → [홈 /home]
```

---

## 관리자 WOD 수정 (`/admin/wod/edit/[id]`)

```
[WOD 수정 /admin/wod/edit/[id]]
    │
    ├── 기존 WOD 데이터 불러오기
    │
    ├── WOD 등록과 동일한 입력 폼
    │
    └── WOD 수정 버튼
            └── 수정 완료 → [홈 /home]
```

## 알림 흐름

```
이벤트 발생
    │
    ├── WOD 등록 (관리자)
    │       └── 모든 유저 → notifications 컬렉션에 문서 생성
    │
    └── 댓글 좋아요
            └── 댓글 작성자 → notifications 컬렉션에 문서 생성

유저가 알림 벨 클릭
    │
    ├── 알림 패널 열림 (슬라이드)
    ├── 모든 알림 읽음 처리
    ├── 알림 목록 표시 (실시간 onSnapshot)
    │       └── 시간 표시: 방금 전 / N분 전 / N시간 전 / N일 전
    └── 읽은 알림 삭제 버튼
```
